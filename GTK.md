# GTK Data 2020

Данные внешнеторговой деятельности Узбекистана за 2020 год.

## Общая информация

- **Файл:** `gtk_data_2020.xlsx`
- **Записей:** 2,917
- **Колонок:** 17
- **Период:** 06.01.2020 — 31.12.2020

## Поля

| Поле | Описание |
|------|-----------|
| Режим | ИМ (импорт) / ЭК (экспорт) |
| Страна | Страна-партнёр |

| Адрес Хориж | Адрес иностранной стороны |

| СТИР Хориж | СТИР иностранной организации |
